# notion palette

> Off-white paper, warm ink, blue accent — soft and purposeful.

| Role | Hex | Use for |
|---|---|---|
| `ink` | `#37352F` | All strokes and body text. The warm near-black Notion uses throughout. |
| `paper` | `#FBFAF7` | The background. Off-white, never pure white — preserves the warm, low-contrast feel. |
| `accent` | `#0F62FE` | Primary emphasis. Callouts, key decisions, the single element you want the eye on. |
| `accent_alt` | `#E16E5B` | Secondary emphasis. Use only when two ideas truly compete side-by-side. |
| `evidence_bg` | `#F1F1EF` | Evidence block backgrounds — code snippets, inline examples. Pale gray, not dark. |
| `evidence_text` | `#37352F` | Text inside evidence blocks. Same as `ink` — evidence stays light-on-light here. |
| `muted` | `#9B9A97` | Labels, grid lines, secondary annotations. |

## Editorial rules

- `paper` is off-white (#FBFAF7), not pure white — this is the Notion aesthetic; don't swap it for #FFFFFF.
- Evidence blocks stay light (`evidence_bg` / `evidence_text` are both light-mode). This theme does not invert for code.
- One `accent` per diagram. `accent_alt` is for true binary comparisons only.
- `muted` carries secondary structure; don't promote it to body copy.
- Italic is used sparingly — treat it as a whisper, not emphasis.
