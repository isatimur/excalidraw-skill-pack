# @excalidraw-skill-pack/theme-stripe-press

> Editorial &#x2F; book-grade. Cream paper, ink-brown strokes, editorial sans&#x2F;serif. Proven on 64 diagrams.

Theme package for [excalidraw-skill-pack](https://excalidraw-skill-pack.dev).

## Install

```bash
npm i -g @excalidraw-skill-pack/theme-stripe-press
# or
pip install excalidraw-skill-pack-theme-stripe-press
```

## Use

```bash
npx @excalidraw-skill-pack/render diagram.excalidraw --theme stripe-press
```

Or set in `.excalidraw-skill-pack.json`:

```json
{ "theme": "stripe-press" }
```
