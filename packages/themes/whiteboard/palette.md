# whiteboard palette

> Pure white, bold black, amber accent — low-fi energy, high-signal structure.

| Role | Hex | Use for |
|---|---|---|
| `ink` | `#1A1A1A` | All strokes and text. Near-black, not quite — keeps the hand-drawn feel. |
| `paper` | `#FFFFFF` | The background. Clean whiteboard surface. |
| `accent` | `#FFB400` | The highlight. Sticky-note yellow energy. Use it to flag the one thing worth acting on. |
| `accent_alt` | `#FF5252` | Second accent — risk markers, blockers, opposing forces. Use it to contrast with amber. |
| `evidence_bg` | `#FFFCEE` | Evidence block fill — pale yellow, keeps the whiteboard warmth. |
| `evidence_text` | `#1A1A1A` | Text inside evidence blocks. Same as `ink`. |
| `muted` | `#999999` | Secondary labels, dashed connectors, stage dividers. |

## Editorial rules

- This theme is for divergent thinking — more accent use is acceptable here than in editorial themes.
- Roughness is exaggerated (roughness: 2) on purpose. Don't smooth it out.
- No italic — the font is Virgil (hand-drawn); italic breaks the visual language.
- `accent` for "this is the cluster that matters"; `accent_alt` for "and this is what blocks it."
- Evidence blocks stay warm (pale yellow fill), not dark — whiteboard sessions stay in light mode.
