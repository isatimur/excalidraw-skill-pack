# @excalidraw-skill-pack/install

One-command installer for excalidraw-skill-pack adapters.

## Usage

```sh
npx @excalidraw-skill-pack/install claude-code
npx @excalidraw-skill-pack/install cursor --lite
```

## Supported adapters

- `claude-code`
- `cursor`
- `codex`
- `gemini-cli`

## Options

| Flag     | Description                        |
| -------- | ---------------------------------- |
| `--lite` | Install the lightweight skill only |

## Platform support

macOS and Linux run a `.sh` script via `bash`. Windows runs a `.ps1` script via `powershell`.
