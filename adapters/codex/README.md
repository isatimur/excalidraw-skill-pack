# Codex adapter

```bash
bash adapters/codex/install.sh       # full (skill + MCP)
bash adapters/codex/install.sh lite  # skill only, no MCP
```

Windows:

```powershell
.\adapters\codex\install.ps1
.\adapters\codex\install.ps1 -Mode lite
```

After install, the skill is available at `~/.codex/skills/excalidraw-diagram/` and the MCP server is registered in `~/.codex/mcp.json`.
