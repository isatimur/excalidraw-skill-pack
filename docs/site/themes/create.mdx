---
title: "Create a theme"
description: "Author + publish a custom theme in 3 minutes."
---

## 1. Scaffold

```bash
npx create-excalidraw-theme my-brand \
  --description "Brand theme for ACME Co." \
  --author "Jane Doe"
cd theme-my-brand
```

## 2. Edit your palette

`palette.json` (machine-readable, used by the renderer):

```json
{
  "ink": "#1a1a1a",
  "paper": "#fff8f3",
  "accent": "#ff5722",
  "accent_alt": "#ffa000",
  "evidence_bg": "#f5f5f5",
  "evidence_text": "#212121",
  "muted": "#9e9e9e"
}
```

`palette.md` (read by the AI agent's system prompt — explain *when* to use each color):

```markdown
| Role        | Hex       | Use for                                  |
|-------------|-----------|------------------------------------------|
| `ink`       | `#1a1a1a` | All text and strokes                     |
| `paper`     | `#fff8f3` | Main background                          |
| `accent`    | `#ff5722` | One per diagram. The argument.           |
| `accent_alt`| `#ffa000` | Secondary accent for comparisons only    |
| `muted`     | `#9e9e9e` | Secondary text, grid lines, dividers     |
```

## 3. Optionally override typography, elements, layouts

Drop files in `elements/box.json`, `layouts/chapter-card.md`, etc. Anything you don't override is inherited from `default-sketchy`.

## 4. Validate

```bash
npx validate-theme theme.json
```

Expected: `OK: theme.json`.

## 5. Render a preview

```bash
npx @excalidraw-skill-pack/render \
  ../../packages/shared/fixtures/03-concept-card.excalidraw \
  --theme my-brand --output preview.png
```

## 6. Publish (npm)

```bash
npm publish --access public
```

Your package name must follow the pattern `@excalidraw-skill-pack/theme-<slug>` for it to appear in the registry.

## 7. Publish (PyPI mirror)

```bash
cd src/excalidraw_skill_pack_theme_my_brand
uv build
uv publish
```

PyPI package name must follow `excalidraw-skill-pack-theme-<slug>`.

## 8. Show up in the registry

Within 24 hours, the [/themes registry](/themes) auto-discovers your package via nightly scan. To get a curated listing, [open a registry-add issue](https://github.com/isatimur/excalidraw-skill-pack/issues/new?template=registry-add.yml).
